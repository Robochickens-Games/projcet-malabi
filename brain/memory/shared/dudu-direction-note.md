# Dudu isn't going as hoped — and that's fine

Dor flagged that the WhatsApp bridge (Dudu) isn't going the way he wanted — the
phone-number and hosting friction turned it into more of a slog than its value
justifies right now. **This is explicitly accepted, not a problem to fix.** No
pressure or urgency to make Dudu happen.

**Update (2026-06-09):** Dudu was removed entirely — the `dudu/` code and the
agent persona are deleted; only the decision trail remains ([[0002-dudu-whatsapp-bridge]],
now `abandoned`). It was noise, not a parked task.

**How to apply:** Don't keep pushing Dudu, and don't treat it as blocked/at-risk
work. If someone ever wants a chat bridge again, start fresh — only when it's
genuinely easy/free. The shared brain is the valuable part and stands on its own.
