_Source: ChatGPT deep research, run by Gidi, 2026-06-11._
_Follows up on [[aso-research-findings]] (2026-06-10). Informs ADR 0003._

---

## Executive summary

**Starfall Fossil Rescue** is the recommended first launch. It fuses dinosaur (strongest
demand) with asteroid/space science — a combination that is less crowded than standalone
dino apps or standalone space apps, and hits a clear market gap: narrative mystery games
that fuse paleontology and astronomy for kids.

Rank:
1. **Starfall Fossil Rescue** — best commercial + educational fit
2. **Time Telescope: Dino to Deep Space** — best episodic pipeline / sequel
3. **Mars Rover and the Last Dino Signal** — best press/teacher angle, narrower store demand

## Cross-theme product rules

| Rule | Recommendation |
|---|---|
| Session length | Design around 5–8 minute scene solves |
| Learning loop | One observable scientific idea per scene, not three |
| Retention | Scientist cards, notebook collections, badges, weekly mini-mysteries |
| Monetization | Free first chapter + premium expansion / parent pass; never lock the main learning spine |
| Parent trust | Clear privacy page, voiceover narration, no third-party ad maze, obvious parent gate |
| Sequelability | Structure each mystery as a reusable "case file" for future content packs |
| Kids Category | Build for Apple Kids Category + Google Play Families / Teacher Approved from day one |

| The House of Da Vinci | 500K+ (Play) | Tactile puzzle-adventure — mechanic reference |

**Market gap:** high-performing titles are dino-only, space-only, or science sandbox.
"Dinosaur + space + mystery + point-and-tap quest" is noticeably thin — good for differentiation.

## Recommended 6-month launch timeline

1. **Month 1:** keyword validation, pick theme, build vertical slice, draft metadata variants
2. **Month 2:** parent + classroom playtests, test 3 screenshot concepts, lock monetization
3. **Month 3:** soft launch limited English markets, Apple Search Ads on long-tail terms, Google test
4. **Month 4:** metadata + onboarding iteration, add localizations, printable experiment pack, creator outreach
5. **Month 5:** full launch, micro-creator content, first post-launch mini-mystery
6. **Month 6:** compound organic reach, second content update, expand winning keyword cluster, decide sequel roadmap

---

## Store metadata strategy

- **App Store:** title / subtitle / keyword field do the heavy lifting
- **Google Play:** title + short + long description — broader semantic coverage in long desc helps
- Lead screenshots: mystery first → scientific action → notebook/collection → real scientist / learning outcome
- Parents must understand learning value within first store screenshots, not only the fantasy wrapper
